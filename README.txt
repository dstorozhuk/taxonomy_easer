A functions set for easy work with taxonomy.

На русском

Краткое описание возможностей:

-Выборка ВСЕХ детей определенного термина термина
-Построение иерархического списка без использования taxonomy_get_children()
-Веб интерфейс для ввода кросс словарной иерархии
-Функции для работы с кросс словарной иерархией
-И другое...

Проект в стадии сборки кода из разработаны модулей в единый модуль для обеспечения вышеперечисленного функционала.
Примеры будут выложены в ближайшее время.
